## 概要

- `NicheLibrary::UInt128` は 128 bit 符号なし整数型である。
- `NicheLibrary::Int128` は 128 bit 符号付き整数型である。
- GCC 拡張には依存しない。
- `std::numeric_limits` に対応している。
- 主な用途は、他のライブラリの内部計算型である。

## 使い方

- `NicheLibrary::UInt128`
  - 128 bit 符号なし整数型である。
  - 整数型から構築できる。
  - 整数型へ明示変換できる。
  - 加算、減算、乗算、除算、剰余、比較を使用できる。
  - 10 進法表記によるストリーム入出力を使用できる。
  - 前提: 入力される値が `NicheLibrary::UInt128` の範囲に収まる。
  - 備考: 演算結果は $2^{128}$ を法として表される。

- `NicheLibrary::Int128`
  - 128 bit 符号付き整数型である。
  - 整数型から構築できる。
  - 整数型へ明示変換できる。
  - 加算、減算、乗算、除算、剰余、比較を使用できる。
  - 10 進法表記によるストリーム入出力を使用できる。
  - 前提: 入力される値が `NicheLibrary::Int128` の範囲に収まる。
  - 前提: 演算結果が `NicheLibrary::Int128` の範囲に収まる。
  - 前提: 除算と剰余では、除数が $0$ でない。
  - 前提: $\min / -1$ および $\min \bmod -1$ を行わない。

- `NicheLibrary::UInt128::from_words(high, low)`
  - 上位 64 bit が `high`、下位 64 bit が `low` である値を返す。

- `NicheLibrary::Int128::from_words(high, low)`
  - 2 の補数表現で、上位 64 bit が `high`、下位 64 bit が `low` である値を返す。

## 計算量

- 加算、減算、乗算、比較: 定数回の 64 bit 整数演算
- 除算、剰余: 128 回程度の 64 bit 整数演算
